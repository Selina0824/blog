使用git所遇到的常见问题

2018-02-26


1. git下载内容过大——增大git postBuffer

报错

git clone https://github.com/cocos2d/cocos2d-x.git
Cloning into 'cocos2d-x'...
remote: Counting objects: 397914, done.
remote: Compressing objects: 100% (19/19), done.
error: RPC failed; result=56, HTTP code = 200 MiB | 0 bytes/s
fatal: The remote end hung up unexpectedly
fatal: early EOF
fatal: index-pack failed

解决方法

使用命令行"git config http.postBuffer 24288000"

查看git postBuffer,使用命令：

"git config --get http.postBuffer"

其他可用命令

git config --global http.postBuffer 524288000

git config --list
